Política de Privacidad

Última actualización: 2026

Esta aplicación respeta tu privacidad. Esta política explica de manera sencilla cómo se maneja la información dentro de la aplicación.

1. Información que recopila la aplicación

Esta aplicación no recopila, almacena ni comparte información personal del usuario.

No se solicitan datos como:

  • Nombre
  • Correo electrónico
  • Ubicación
  • Contactos
  • Información del dispositivo

2. Almacenamiento local

La aplicación utiliza almacenamiento local del dispositivo para guardar configuraciones del temporizador y preferencias del usuario.

Estos datos se guardan utilizando SharedPreferences, un sistema estándar de almacenamiento local en Android.

Los datos pueden incluir:

  • Duración del temporizador Pomodoro
  • Duración de descansos cortos
  • Duración de descansos largos
  • Preferencias de configuración de la aplicación

Toda esta información:

  • Se guarda únicamente en el dispositivo
  • No se envía a servidores
  • No se comparte con terceros

3. Uso de internet

La aplicación no requiere conexión a internet para funcionar y no transmite datos a servidores externos.

4. Servicios de terceros

La aplicación no utiliza servicios de terceros que recopilen información del usuario.

5. Cambios en esta política

Si en el futuro la aplicación cambia su funcionamiento, esta política será actualizada para reflejar dichos cambios.

6. Contacto

Si tienes preguntas sobre esta política de privacidad, puedes contactar al desarrollador a través de la página oficial de la aplicación.
